# sacc-team-A

Welcome to Sacc Team A Project

## To install 

```npm install```

```gcloud config set project sacc-team-a```

## To run at local

```npm run dev```

## To deploy

```npm run deploy```

## Emulator datastore

### Prepare 

```
export DATASTORE_EMULATOR_HOST=localhost:8081
export DATASTORE_PROJECT_ID=sacc-team-a
export DATASTORE_USE_PROJECT_ID_AS_APP_ID=true
```

### Run emulator

```
npm run emulator
```
